# Daily Feed Project Summary

Daily Feed is an automated academic paper discovery and summarization tool written in Go. It provides a flexible pipeline for researchers and knowledge workers to stay updated with the latest academic literature.

## Key Features
- Fetch recent academic papers from arXiv
- Intelligent paper selection
- AI-powered paper summarization using Anthropic Claude
- Configurable publication methods (stdout, email, web)
- Customizable search topics and scheduling

## Technical Overview
- Language: Go (1.25+)
- Dependencies: Anthropic Claude API
- Configurable via YAML
- Multiple publication modes
- Cron-based scheduling

## Use Cases
- Academic research monitoring
- Technology trend tracking
- Personal knowledge management
- Automated literature review

## Configuration Options
- Search topic selection
- Paper quantity limits
- Digest schedule
- Publication method
- Immediate startup option
